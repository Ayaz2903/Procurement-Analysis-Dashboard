# Procurement Analysis Dashboard (Power BI Data Analysis Project)

## Project Objective

The objective of this project is to analyze procurement and purchase order data to identify spending patterns, evaluate supplier performance, monitor order fulfillment, and support data-driven procurement decisions. The dashboard provides actionable insights that help improve operational efficiency and optimize procurement processes.

## About the Project

This project provides a complete Power BI procurement dashboard designed to track purchasing activities, supplier performance, and procurement costs through interactive visualizations.

### Questions Solved:

1. What is the total procurement spend?
2. Which suppliers perform the best?
3. What is the distribution of purchase order statuses?
4. Which product categories have the highest procurement costs?
5. How can procurement efficiency be improved?
